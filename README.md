## 🎮 游戏简介

Text WoW 是一款放置类文字RPG游戏，玩家制定策略，角色自动战斗成长。

**核心特点：**
- 🖥️ DOS终端复古风格界面（黑底绿字、CRT扫描线效果）
- ⚔️ 自动回合制战斗
- 🧠 策略制定（而非手动操作）
- 🌍 魔兽世界风格的种族和职业

## 📋 技术栈

| 组件 | 技术 |
|-----|------|
| 前端 | Vue 3 + TypeScript + Vite |
| 后端 | Go + Gin |
| 数据库 | SQLite (WAL模式) - *原型暂未使用* |

## 🚀 快速开始

### 环境要求

- Go 1.21+
- Node.js 18+
- pnpm / npm / yarn

### 1. 初始化数据库（首次运行或重置数据库后）

```bash
cd server
go run cmd/init_database/main.go
```

这会自动：
- 创建数据库表结构
- 加载基础数据（区域、怪物等）
- 加载战士技能数据
- 添加杜隆塔尔区域怪物

### 2. 启动后端

```bash
cd server
go mod tidy
go run main.go
```

后端将在 `http://localhost:8080` 启动

### 3. 启动前端

```bash
cd client
npm install
npm run dev
```

前端将在 `http://localhost:5173` 启动

### 4. 开始游戏

浏览器访问 `http://localhost:5173`

## 🎯 游戏玩法

### 角色创建
1. 输入角色名称
2. 选择种族（联盟/部落）
3. 选择职业（战士/法师/盗贼/牧师）

### 自动战斗
- 点击 **[▶] 开始挂机** 开始自动战斗
- 角色会自动攻击当前区域的怪物
- 击败怪物获得经验和金币
- 经验满了自动升级

### 控制按钮
| 按钮 | 功能 |
|-----|------|
| [▶] 开始挂机 | 开始/停止自动战斗 |
| [→] 单步战斗 | 手动执行一个回合 |
| [S] 策略 | 战斗策略配置（待开发） |
| [E] 装备 | 装备管理（待开发） |
| [M] 地图 | 区域选择（待开发） |
