## 🎯 快速参考

### 伤害计算流程

详细公式请参考 [数值计算系统设计](./calculator_design.md)

```
1. 基础伤害 = 攻击力 × 技能系数
2. 防御减伤 = 基础伤害 × (1 - 防御减伤率)
3. 暴击判定 = 如果暴击，伤害 × 暴击倍率
4. 闪避判定 = 如果闪避，伤害 = 0
5. 最终伤害 = max(1, 最终伤害)
```

### 仇恨系统

详细设计请参考 [战斗系统设计](./battle_system_design.md)

- **基础仇恨** = 伤害量 × 仇恨系数
- **治疗仇恨** = 治疗量 × 0.5（分摊给所有敌人）
- **嘲讽** = 强制成为第一仇恨

### 资源系统

详细设计请参考 [技能系统设计](./skill_system_design.md)

| 资源类型 | 获取方式 | 消耗方式 |
|---------|---------|---------|
| 怒气 | 攻击/受击获得 | 技能消耗 |
| 法力 | 回合恢复 | 技能消耗 |
| 能量 | 回合恢复（快） | 技能消耗 |
